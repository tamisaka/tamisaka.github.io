box 

box(x,y,w,h)

概要

E().box()は、spanによる四角形を描くメソッドです。描画されたspanはE()で取得したエレメントの下に追加されます。

引数 内容 型 必須
x 左上x座標 Number yes
y 左上y座標 Number yes
w 横幅 Number yes
h 縦幅 Number yes

解説

通常、Javascriptを用いてブラウザ上で四角を描くにはcanvasやSVGがよく使われます。

//canvasの場合
var cv=document.getElementById("canvas1"),
ctx=cv.getContext('2d');
ctx.fillStyle='red';
ctx.fillRect(25,25,50,50)

//SVGの場合
var sv=document.getElementById("svg1"),
rect=document.createElementNS("http://www.w3.org/2000/svg","rect");
rect.setAttribute('x',50);
rect.setAttribute('y',50);
rect.setAttribute('w',50);
rect.setAttribute('h',50);
rect.setAttribute('fill','red');
rect.setAttribute('stroke','black');
sv.appendChild(rect);

canvasでは比較的容易にコーディング可能ですが、後から変形したりする事が難しくなっています。

SVGで円を描く場合は親要素がsvgであることが前提です。また、少なくとも２つ以上のタグを必要とするほか、シンプルな図形を描くためのコードの行数が多くなってしまいます。

kentaur.jsでは生成される新たなspan要素に対して自動的に必要なCSSスタイルを適用し、以下の通り１行にまとめて書くことができます。

E("div#kentaur1").box(50,50,50,50)

上記の場合、横50px,縦50pxの座標から横50px,縦50pxの範囲内で四角形を描きます。縦横の幅が一致していない場合は長方形になります。
