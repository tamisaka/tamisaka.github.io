E()

E(element)

引数 内容 型 必須
element 取得したい要素のCSSセレクタ
またはElementそのもの		 String
Object		 yes

E()は、Kentaur.jsにおけるHTMLから任意の要素を取得するためのセレクタです。CSSセレクタで取得したい要素を選択します。

E("body");
E(document.body);
E("div#target");

document.querySelectorと使い方はほとんど同じですが、返り値にはエレメントが直接取得されるのではなく、取得した要素の子要素として線を描画したり、取得した要素自体をアニメーション移動させるための様々な関数が渡されます。
