line

概要


E().line()は、spanによる直線を描くメソッドです。描画されたspanはE()で取得したエレメントの直下に追加されます。

line(x1,y1,x2,y2,speed)
引数 内容 型 必須
x1 始点のx座標 Number yes
y1 始点のy座標 Number yes
x2 終点のx座標 Number yes
y2 終点のy座標 Number yes
speed 速度(ms) Number no

解説


通常、Javascriptでブラウザ上で直線を描くには、canvasやSVGがよく使われます。


    //canvasの場合
    var cv=document.getElementById("canvas1"),
    ctx=cv.getContext('2d');
    ctx.moveTo(100,50);
    ctx.lineTo(200,250);
    ctx.closePath();
    ctx.stroke();

    //SVGの場合
    var sv=document.getElementById("svg1"),
    line=document.createElementNS("http://www.w3.org/2000/svg","line");
    line.setAttribute('x1',100);
    line.setAttribute('y1',50);
    line.setAttribute('x2',200);
    line.setAttribute('y2',250);
    line.setAttribute('stroke','black');
    sv.appendChild(line);

しかし、canvasでは一度描画した図形のみを後から変形することはできず、画面を再描画する事によって実現されます。SVGでは後からDOM操作も可能ですが、親となるsvgタグと合わせて少なくとも2つ以上のタグが必要となります。

Kentaur.jsではspanのstyle属性下に対して必要なCSSスタイルを適用するので、単一のspanタグによって描画することができます。すなわち、以下の通りに書くことができます。

E(“div#kentaur1”).line(100,50,200,250);

また、第5引数に整数値(ms)を入力することで、始点から伸びていくようなアニメーションを伴って生成させることもできます。

サンプルを見る 

E(“div#kentaur1”).line(0,0,1000,700,3000);

もちろん、描画されたものはspanタグそのものであり、後からこのspanタグに対してDOM操作を行い、変形させたり色を変更する事も可能です。

座標の指定はパーセンテージでの指定も可能ですが、線の長さは現在のウィンドウの解像度（画面サイズ）に基づいて計算された数値が設定されます。
